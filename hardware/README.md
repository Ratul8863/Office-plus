# Hardware

This folder holds the physical / simulated side of OfficePulse HQ. The
only current target is a Wokwi-simulated ESP32 DevKit C v4 acting as
the **Work Room 1** device controller.

## Sub-folders

| Path                          | Purpose                                                       |
|-------------------------------|---------------------------------------------------------------|
| `wokwi/`                      | ESP32 firmware + diagram + payload spec for Work Room 1.      |

See `wokwi/README.md` for a 3-step "open in Wokwi, import diagram,
paste firmware" walkthrough.
